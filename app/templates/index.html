Automated Time Series Forecasting API

This is a simplified version of the backend developed for the capstone project titled "Automated Time Series Forecasting: Using Hybrid Forecasting Approach". Refer to this link for the extended abstract. You can read the full document and explore my other projects.

The original system was built using PHP's Laravel framework and Python's Flask for forecasting logic. This version separates concerns into:

  • Frontend: React.js
  • Backend: Python Flask (lightweight and fast)

API Endpoints

  • / – API documentation
  • /api/system-info – Information about the running application
  • /api/preprocess/handle-missing-values – Handle missing values in a univariate time series
  • /api/preprocess/check-series – Validate the time series format
  • /api/forecast – Returns forecasting model information
  • /api/forecast/univariate – Performs n-step forecasting using a hybrid model
